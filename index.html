Welcome to my page.

Hi, thanks for visiting my page. As you have probably guessed, my name is Alex. I'm just an oridnary guy emerging in the world of web development.

After years of working as an engineer, I decided it was time to pursue a new challenge. So I took it upon myself to learn the fundamentals of web development. Now I spend my days creating web apps using a variety of tools including JavaScript, HTML5, CSS, jQuery and modern web technologies.

Shopping List

A To-do list type of app, which uses JQuery, and HTML5 for retaining list items across the page.

Number Guess

This app uses JQuery and HTML5 to randomly select a number from 1 to 100. The app continously compares your guess to the hidden secret number and provides a feedback on how close ("hot") or far ("cold") your guess is.

MovieScape

In addition to JQuery and HTML5, this app uses AJAX to help you search movies by title and it returns information about that movie, including reviews, synopsis(if available), and images.
